# Project Notes

## Deployment
- **Backend**: Deployed on Railway. Auto-deploys from `main` branch on GitHub.
- **Mobile**: Uses EAS (Expo Application Services) for builds and app store submission.
- **Smart Contract**: Solana/Anchor, deploy scripts in `smart-contract/package.json`.

## Tech Stack
- Backend: Node.js, Express, Socket.IO, PostgreSQL
- Mobile: React Native / Expo
- Database migrations: SQL files in `backend/database/migrations/`
